# 개요

특정 기념일로부터 얼마나 지났는지 일수(Day)를 보여주는 앱을 만든다.   
날짜를 핸들링하기 위한 플러터의 `DateTime 타입`과 날짜 선택 관련 UI를 위한 `CupertinoDatePicker 위젯`을 다뤄본다.   
또한, 변화하는 값인 날짜를 `state`로 관리한다.
